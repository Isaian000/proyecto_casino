# Proyecto Casino

Servidor HTTP desarrollado con Node.js, Express y TypeScript, conectado a MongoDB Atlas.
Actualmente cuenta con la configuración base del servidor y la conexión a base de datos.
Próximamente se implementarán los juegos de 21 y ruleta.

## Cómo levantar el servidor

### 1. Instalar dependencias
```bash
npm install
```

### 2. Modo desarrollo
```bash
npm run dev
```

### 3. Modo producción
```bash
npm run build
npm start
```

El servidor estará disponible en `http://localhost:3000`
